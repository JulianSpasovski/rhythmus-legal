# Rhythmus Legal and Support Site

This folder contains the static public pages for Rhythmus privacy and support.

It is intended to be copied into the separate public GitHub Pages repository `JulianSpasovski/rhythmus-legal`.
Do not publish the private app repository.

## Files

- `index.html`
- `privacy.html`
- `support.html`
- `styles.css`
- `.nojekyll`

The site uses no JavaScript, cookies, analytics, forms, external fonts, external scripts, CDN assets, images, phone number, or physical address.

## Publishing Steps

1. Create a separate public GitHub repository named `rhythmus-legal`.
2. Copy only these site files into that repository.
3. Commit and push to the `main` branch.
4. Open GitHub repository settings, then Pages.
5. Set Source to `Deploy from a branch`.
6. Set Branch to `main`.
7. Set Folder to `/root`.
8. Save.
9. Verify these URLs in a private/incognito browser:
   - `https://julianspasovski.github.io/rhythmus-legal/`
   - `https://julianspasovski.github.io/rhythmus-legal/privacy.html`
   - `https://julianspasovski.github.io/rhythmus-legal/support.html`

## Publication Gate

The app code and submission docs should point to `https://julianspasovski.github.io/rhythmus-legal/` after GitHub Pages is live.
Do not archive, upload, or mark the Apple readiness checklist complete unless the public Pages URLs are reachable without login.
